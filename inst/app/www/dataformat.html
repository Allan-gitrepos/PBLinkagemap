PBlinkagemap Data Format

Data Format

PBlinkagemap requires your data to be in a specific format in order to perform the QTL analysis and mapping. The following table shows the required format:

Column Description Data Type
Marker The name of the marker. String
Chromosome The chromosome number. Integer
Position The position of the marker on the chromosome in centimorgans (cM). Float
Trait The trait that you are interested in mapping. Float


Please note that all columns in the table must be present and in the correct order. You can save your data in a CSV file and then import it into PBlinkagemap.
